更新WordPress的登录屏幕使用高分辨率，全屏，缩放背景图像。现在已经完全响应。

==描述==



该品牌登录屏幕插件的特点：

*全屏幕背景图像
*使用自己的标题图片在WordPress的LOGO位置
*位置登录表单：左对齐，居中或右/上，中
*登录表单响应小屏幕
* SCROLLING本，如果超出视口
*背景图片大小随窗口调整大小
*支持重复背景图像
*删除所有WordPress的商标标识
*静止升级预览



==常见问题==


= Q）如何更改登录表单位置您网站上的登录页面？ =
A）转到插件页面在网站中的管理部分。

找到“品牌登录屏幕”插件条目，然后选择“编辑” 。这将带您进入“编辑插件”屏幕的“品牌 - 登录 - screen.php ”加载并可以进行编辑。

向下滚动并查找以下行：

    48限定（' BLS_LOCATION '，0） ;

在插件的未经编辑的版本，它是第48行。

更改该值与您想要的位置，以协调的登录表单。

* 0 - 中东中心
* 1 - 中间偏左
* 2 - 中 - 右
* 3 - 顶部中心
* 4 - 左上
* 5 - 右上
 
在下面的编辑文本框的“更新文件”按钮。
检查可湿性粉剂登录页面，以确保变更已影响。

= Q） HOW TO：更改登录标题图片？ =
A）转到插件页面在网站中的管理部分。

找到“品牌登录屏幕”插件条目，然后选择“编辑” 。这将带您进入“编辑插件”屏幕的“品牌 - 登录 - screen.php ”加载并可以进行编辑。

向下滚动并查找以下行：

    53限定（' BLS_HDR_LOGO '，' header.png '）;
    54定义（' BLS_RHDR_LOGO '，'标题 - sm.png '） ;

在插件的未经编辑的版本，它是线53及54 。

这两个设置是全尺寸头徽标图像（ BLS_HDR_LOGO ）和响应报头标志图片（ BLS_RHDR_LOGO ） 。

全尺寸的标头是340px X 87px （宽x高） 。你的形象并不一定是因为宽，但如果是更高的底部将被裁剪。

该响应报头是一样的高度，但仅仅是260px宽。同样地，如果它是高或更宽它会被裁剪。

最好是让2个图像，每一个模式虽然如果得到适当的布局， 1图像可以工作两种状态。如果您决定使用一个图像，你将不得不作出2份，并在适当地点的文件夹。

全尺寸头部logo图片应该放在以下文件夹：
    ' / branded-login-screen/assets/i/logo_main / '

该响应报头标志形象应放在以下文件夹：
    ' / branded-login-screen/assets/i/logo_responsive / '

如果图像名称不是默认的名称（ header.png与头 - sm.png ） ，您将需要编辑线53及以上用适当的名称提到的54 。

在下面的编辑文本框的“更新文件”按钮。
检查可湿性粉剂登录页面，以确保变更已影响。

= Q） HOW TO：更改背景图片？ =
A）转到插件页面在网站中的管理部分。

找到“品牌登录屏幕”插件条目，然后选择“编辑” 。这将带您进入“编辑插件”屏幕的“品牌 - 登录 - screen.php ”加载并可以进行编辑。

向下滚动并查找以下行：

    50定义（' BLS_FULL_SCREEN ' ， TRUE） ;
    51
    52定义（' BLS_BG_IMG '，'默认img.jpg '） ;

如果您使用的是全屏幕的背景图像简单地上传新的图片到插件'/资产/ I / background_image “文件夹。如果图像有不同的名称比线52 （默认img.jpg ） ，编辑线52 ，以匹配您上传的图像名称。

如果您使用的是可重复的图像为背景，上线50变化时真亦假（不包括引号） 。当然，这需要新的图像上传到了“ /资产/ I / background_image ”文件夹和行52的编辑，如果图像名称是不一样的在该行的图像名称。

在下面的编辑文本框的“更新文件”按钮。
检查可湿性粉剂登录页面，以确保变更已影响。
