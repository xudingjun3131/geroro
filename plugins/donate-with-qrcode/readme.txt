Donate With QRcode是一款支持使用支付宝、微信支付和PayPal的文章打赏插件。

== Description ==

Donate With QRcode开发的初衷是方便WordPress博主在文章末提供支付宝、微信支付和PayPal多个打赏渠道，方便读者打赏（捐赠）站长以鼓励站长继续创作贡献。

其中支付宝和微信支付支持使用扫码支付，PayPal则需要通过链接支付。


在使用Donate With QRcode插件前，你首先需要到 “WordPress 后台-插件-已安装的插件”，找到插件 Donate With QRcode，点击启动；然后点击“设置”或者通过 “WordPress 后台-插件-Donate With QRcode”对插件进行配置：


== Installation ==

1. 上传 `donate-With-qrcode` 目录到 `/wp-content/plugins/`目录.
2. 在 WordPress 插件面板激活 `Donate With QRcode` 插件.
3. 通过设置->`Donate-With-QRcode` ，上传微信和支付宝二维码，及设置PayPal捐赠链接即可。
